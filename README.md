# Finance Dashboard Project

Dashboard ini dibuat untuk menganalisis data keuangan pribadi secara dinamis menggunakan Microsoft Excel. Proyek ini mendemonstrasikan kemampuan visualisasi data dan pengolahan data menggunakan Pivot Tables.

## Fitur Utama
- **Interaktif:** Menggunakan Slicers untuk memfilter data berdasarkan bulan secara real-time.
- **Custom Formatting:** Implementasi notasi angka ribuan (k) untuk kemudahan pembacaan data.
- **Visualisasi:** Grafik tren pengeluaran dan kategori yang terintegrasi dengan Pivot Table.
- **Design:** Tema "Old Money" (Navy & Gold) dengan logo kustom "The Dyva Dash".

## Tools yang Digunakan
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers & Timeline
- Custom Number Formatting
